User
====

Overview
--------

A user represents someone who can login to the website administrator.
See :doc:`/devguides/permissioning` for more information on users and permissioning.

**Object name:**
    security_user

**Table name:**
    psys_security_user

**Path:**
    /preside-objects/admin/security/security_user.cfc
